## Module 37: Advanced - Building a Personalized Shopping Agent

## Overview

### Agent Architecture

This advanced project is a **distributed multi-agent system** composed of three specialized agents communicating via A2A:

1.  **Orchestrator Agent:** The main, user-facing agent that manages the conversation and delegates tasks.
2.  **Personalization Agent:** A remote agent for managing user preferences using persistent state.
3.  **Web Agent:** A remote agent that acts as an interface to a simulated e-commerce website, exposing tools for searching and clicking.

This capstone project combines concepts from across the entire course, including multi-agent architecture, A2A communication, state management, and tool use.
