## Module 29: Introduction to UI Integration

## Overview

### The UI Integration Landscape

The ADK supports several approaches for UI integration:

| Approach | Best For |
| :--- | :--- |
| **AG-UI Protocol** | Modern web apps (React/Next.js) |
| **Native ADK API** | Custom frameworks (Vue, Angular) |
| **Direct Python** | Data apps (Streamlit) |

### Understanding the AG-UI Protocol

**AG-UI (Agent-Generative UI)** is an open protocol for agent-user interaction, developed through a partnership between the Google ADK and CopilotKit teams. It provides a standardized, event-based way for AI agents to communicate with web UIs, enabling you to quickly build a production-ready chat interface with pre-built components.
